## Features

- **Increment** - Add 1 to the counter
- **Decrement** - Subtract 1 from the counter
- **Reset** - Set counter back to 0

## Getting Started

```bash
# Compile the app
jnc compile main.jnc

# Run development server
jnc dev

# Open http://localhost:3000
```

## What You'll Practice

- Creating and using signals
- Writing event handlers
- Defining functions in components
- Using utility classes for styling

## Customize It!

Try these challenges:

1. **Add a step size** - Let users increment by 5, 10, etc.
2. **Add min/max limits** - Prevent going below 0 or above 100
3. **Add undo/redo** - Track history and let users go back
4. **Make it pretty** - Add animations when count changes
