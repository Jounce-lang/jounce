## Features

**Components**:
- `StatCard` - Reusable stat display with trend indicator
- Tab-based navigation
- Activity timeline
- Action buttons

**Layout**:
- Navigation bar
- 4-column grid for stats
- 2-column grid for content
- Responsive container

**State Management**:
- Tab selection with signals
- Stats array management
- Conditional rendering per tab

## Getting Started

```bash
# Compile the app
jnc compile main.jnc

# Run development server
jnc dev

# Open http://localhost:3000
```

- Use consistent spacing (multiples of 4)
- Stick to 2-3 colors max
- Use shadows for depth
- Add loading states
- Include empty states
